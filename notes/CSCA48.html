Bitwise Operators

Everything in a computer is represented by bits. What are bits?

2 = 00000010
7 = 00000111
64 = 01000000

Example code:

int a = 2;
int b = 7;
int c = a&b;

What does & do? & is known as one of many bitwise operators; this one being specifically the AND operator.

Logical Operators

a=2;  
b=7;  
c=a&b;

if (a==2 && b==7)  
{  
	printf("A is 2 and b is 7\n");  
}

Remember, logical expressions are evaluated to be TRUE or FALSE.
Which of the above are logical expressions?

They are:

a==2;
b==7;
a==2 && b==7

Summary of Operators

Logical Operators:

&& - AND
|| - OR
== - EQUALS
> - GREATER
>= - GREATER OR EQUAL

Bitwise Operators
& - AND
| - OR

Note: Just like Python, == is a logical comparison and = is for assignment. However, C allows you to run things like the following example.
