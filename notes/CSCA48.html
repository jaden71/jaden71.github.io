Bitwise Operators

Everything in a computer is represented by sequences of bits. What are bits?

Example:

2 = 00000010
7 = 00000111
64 = 01000000

int a = 2;
int b = 7;
int c = a&b;

What does & do? & is known as one of many bitwise operators; this one being specifically the AND operator.

Logical Operators

Logical operators were introduced in CSCA08 with Python.

a && b is equivalent to a and b
a || b is equivalent to a or b
!a is equivalent to not a

a=2;  
b=7;  
c=a&b;

if (a==2 && b==7)  
{  
	printf("A is 2 and b is 7\n");  
}

Remember, boolean expressions are expressions that are evaluated to be TRUE or FALSE.

Example

Things to take note of:

  • Just like Python, == is a logical comparison and = is for assignment. However, C allows you to run things like the following example.
  • An assignment returns the rightmost value in the assignment.
  • The precedence of bitwise operators is higher than logical operators. For a complete list of precedence, consult the documentation.
int a=2;  
int b=0;  
int c=7;  
  
if ((a=b && (c=3) & b==0)) 
{  
	printf("Is this going to get printed?\n"); 
} 
else 
{  
	printf("Or is this going to get printed?\n"); 
}
