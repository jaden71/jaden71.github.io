What are the advantages of calloc over regular arrays?

Example 1
Consider the following code:

int *newArray(int size)  
{  
	int *array; // A pointer! not an array declaration  
	array=(int *)calloc(size,sizeof(int)); // Get 'size' items large

Memory Model Review

Key facts to remember:

  • Every time a variable is declared, a “box” is allocated to store information. That allocated location in memory is then stored.
  • Variables are local within the function they are declared in, so each function has its own space in memory.
